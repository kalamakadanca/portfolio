document.getElementById('contact-form').addEventListener('submit', function(event) {
  event.preventDefault(); // Zabráníme standardnímu odeslání formuláře

  const name = document.getElementById('name').value;
  const email = document.getElementById('email').value;
  const message = document.getElementById('message').value;

  // Validace (např. kontrola, že všechna pole jsou vyplněná)
  if (name && email && message) {
    // Zobrazíme potvrzovací zprávu
    document.getElementById('confirmation-message').style.display = 'block';

    // Vyčistíme formulář po odeslání
    document.getElementById('contact-form').reset();
  } else {
    alert('Prosím, vyplňte všechna pole!');
  }
});


To je skvělý nápad! Pro tmavý režim, který bude příjemný pro oči, je dobré použít tmavší odstíny, ale ne čistě černou, protože ta může být příliš kontrastní a unavující pro oči při dlouhém sledování. Zaměříme se tedy na barvy, které jsou populární v designu pro programování, grafiku a matematiku, kde je kladen důraz na příjemný kontrast a pohodlné čtení.

Doporučené barvy pro tmavý režim:
Pozadí:

Tmavá šedá (#1e1e1e nebo #2d2d2d) místo čisté černé – méně náročné na oči.

Tmavě modrá (#1a1f2d) – modrá barva je často spojována s programováním a technologiemi a má příjemný vizuální efekt.

Text:

Světle šedá (#e1e1e1 nebo #d3d3d3) – pro běžný text, aby byl čitelný, ale stále jemný pro oči.

Bílá (#ffffff) – pro důrazné texty nebo nadpisy.

Světle modrá (#66c0f4) nebo smaragdová (#76e5b3) – pro akcenty, odkazy nebo tlačítka, což přidává živost, ale není příliš intenzivní.

Akcentní barvy:

Oranžová (#ff9f00) nebo zelená (#5cb85c) – pro interaktivní prvky (tlačítka, odkazy) a pro zvýraznění.

Fialová (#9b59b6) – pro příjemný kontrast a vizuální zajímavost.

Žlutá (#f1c40f) – na zvýraznění důležitých informací nebo upozornění.

Příklad barevné palety:
Pozadí: Tmavě šedá (#2d2d2d), tmavá modrá (#1a1f2d)

Text: Světle šedá (#e1e1e1), bílá (#ffffff)

Odkazy a tlačítka: Světle modrá (#66c0f4), oranžová (#ff9f00)

Akcenty: Zelená (#76e5b3), fialová (#9b59b6), žlutá (#f1c40f)

Tato paleta by měla být ideální pro vizuálně přitažlivý, ale zároveň komfortní vzhled pro oči, který bude ideální pro tvé portfolio zaměřené na oblast programování a grafiky.

Doporučení pro implementaci:
CSS pro tmavý režim: Pomocí mediálních dotazů můžeš nabídnout podporu pro tmavý režim (pokud uživatel má nastavený tmavý režim na svém zařízení), nebo nastavíš tmavý režim jako výchozí a umožníš uživateli přepínat.

Interaktivní prvky: Pro formuláře, validace a přepínání témat doporučuji používat decentní akcentní barvy (např. světle modrá na interakce).

Responsivní design: Můžeš použít CSS Flexbox nebo Grid pro responzivní layout, aby byla aplikace dobře přizpůsobená různým zařízením.
